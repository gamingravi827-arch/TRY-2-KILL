# TRR TO KILL (T2K) Guild Website

Me oya illa a "Supiri" Free Fire Guild Website eka! (Here is the Super Free Fire Guild Website you asked for!)

This is a premium, multi-page website built with HTML and Tailwind CSS. No Node.js is required.

## How to run (Run karana widiha):
Simply double-click the `index.html` file to open it in your browser (Chrome, Edge, etc.).

## Pages Included:
1. **Home (index.html)**: Main page with hero section, stats, and features.
2. **About (about.html)**: Guild history and rules list.
3. **Squad (members.html)**: Leader, Co-leaders, and Elite Squad members.
4. **Gallery (gallery.html)**: Screenshots and best moments.
5. **Join Us (join.html)**: Recruitment form to join the guild.

## Customization:
- You can edit the text in the `.html` files using Notepad or VS Code to change names, rules, or links.
- The design uses a Dark/Red theme typical for Esports.
